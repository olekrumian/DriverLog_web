## Image Guidelines

- **Format**: JPG or PNG (JPG recommended for photos)
- **Dimensions**: 400-600px wide for mobile screenshots
- **Aspect Ratio**: Keep original mobile aspect ratio (roughly 9:19 or similar)
- **File Size**: Optimize/compress images (aim for <200KB each)
- **Quality**: High enough to be clear but optimized for web

## Tools for Image Optimization

- [TinyPNG](https://tinypng.com/) - Free online image compression
- [ImageOptim](https://imageoptim.com/) - Mac app for image optimization
- [Squoosh](https://squoosh.app/) - Web-based image optimizer by Google
